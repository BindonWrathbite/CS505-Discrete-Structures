# CS505-Discrete-Structures
Repository for CS 505: Discrete Structures coursework at SNHU. Includes assignments, activities, proofs, and projects covering logic, set theory, combinatorics, graph theory, and algorithmic problem solving using discrete mathematics.

## Repository Contents
- Activities – Weekly assignments focused on applying discrete mathematics concepts  
- Discussions – Written responses connecting theory to real-world scenarios  
- Short Papers – Analysis of discrete structures and their applications  
- Project – Final project integrating course concepts  
- Journal – Reflection on learning and professional growth  

## Tools and Technologies
- Git and GitHub for version control  
- Markdown for documentation  
- Additional tools as required by specific assignments  

## Notes
- All work in this repository is created for academic purposes  
- Assignments are uploaded as they are completed throughout the course  
- This repository serves as both a portfolio and a record of progress
